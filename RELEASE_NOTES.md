# Clean.me Release Notes

## What's New

### Features
- One-click PC cleanup with beautiful glassmorphism UI
- Comprehensive cleanup of 60+ locations
- Safe operations - never touches personal files
- Real-time progress feedback
- Worker thread architecture for smooth performance

### What Gets Cleaned
- ✅ Windows temporary files and cache
- ✅ Browser data (Chrome, Edge, Firefox)
- ✅ Application cache (Teams, Slack, Zoom, Spotify, Discord)
- ✅ System logs and crash dumps
- ✅ Recycle bin and Windows Update cache
- ✅ Prefetch files

### Installation Options

**Installer (Recommended)**
- Download `Clean.me Setup [version].exe`
- Run with administrator privileges
- Follow the installation wizard

**Portable Version**
- Download `Clean.me-Portable-[version].zip`
- Extract to any folder
- Run `Clean.me.exe` with administrator privileges

**Source Code**
- Download `Clean.me-Source-[version].zip`
- Requires Node.js 16+ to build
- See README.md for build instructions

## System Requirements
- Windows 10 or later (64-bit)
- Administrator privileges for full cleanup
- 100MB RAM minimum
- 200MB disk space for installation
